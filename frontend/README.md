# Frontend - Sales Dashboard

This is the frontend of the Sales Management System built using **React** and **Tailwind CSS**.

It connects to the backend API to display a dashboard with:
- Search
- Filters
- Sorting
- Pagination
- Summary cards
- Sales data table

## Setup Instructions

1. Go to the frontend folder:

```bash
cd frontend

npm install

npm run dev

The app will run at: http://localhost:5173
